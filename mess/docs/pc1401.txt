Sharp Pocket Computer 1401
--------------------------

small scientific calculator and pocket computer


Features
--------
16 digit (5x7 pixel) lcd display
integrated speaker
connector for printer and tape drive
2 "build in" batteries


Error Codes
-----------
1 syntax error
2 calculation error
3 illegal function argument
4 too large a line number
5 next without for
  return without gosub
6 memory overflow
7 print using error
8 i/o device error
9 other errors


Mess Emulation State
--------------------
incomplete cpu core
speaker not emulated
tape not emulated
printer not emulated
work stopped until 
 more "sharps" are dumped 
 info about cpu power/interrupt management is available
